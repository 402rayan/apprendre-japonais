# Grammaire - Points appris

## Structure de base

Le japonais suit l'ordre : **Sujet - Objet - Verbe**

Exemple : わたしは すしを たべます
- わたしは = Je (sujet + particule は)
- すしを = sushi (objet + particule を)
- たべます = mange (verbe)

## Particules essentielles

| Particule | Rôle | Exemple |
|-----------|------|---------|
| は (wa) | Marque le sujet/thème | わたし**は** = "moi, je..." |
| を (wo) | Marque l'objet direct | すし**を**たべます = "mange du sushi" |
| に (ni) | Direction, destination, heure | えき**に**いきます = "vais à la gare" |
| で (de) | Lieu d'action, moyen | でんしゃ**で** = "en train" |
| か (ka) | Question | すしです**か**? = "c'est du sushi ?" |
| の (no) | Possession | わたし**の** = "mon/ma" |
